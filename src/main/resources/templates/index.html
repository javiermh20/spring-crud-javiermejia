    CREATE

  • Esta funcionalidad nos permite crear nuevos elementos a nuestras tablas en la base de datos, asi como tambien nos servira para llevar el registro de clientes, ventas, empleados o algun objeto especifico.

    • READ

  • Esta funcionalidad nos permite ver los elementos que ya estan en nuestras tablas en la base de datos nos permite visualizar a detalle los registros que ya se encuentran guardados en nuestra BD.

    • UPDATE

  • Esta funcionalidad nos permite actualizar los elementos que ya estan en nuestras tablas en la base de datos sirve por si hay una modificación, de cualquier parte o alguna equivocación en la creación.

    • DELETE

  • Esta funcionalidad nos permite eliminar los elementos que ya estan en nuestras tablas en la base de datos sirve como su traduccion en ingles, la podremos utilizar cuando haya registros viejos que no queremos mas en nuestras BD.
